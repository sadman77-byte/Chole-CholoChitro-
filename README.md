# 🎬 Chole Cholochitro (Moving Pictures)

## 📌 About The Project
**Chole Cholochitro** is a unique, massively aggregated legal streaming platform designed for true cinephiles and knowledge seekers. This application entirely bypasses mainstream trending content and focuses exclusively on classic, timeless, and alternative media. 

Our goal is to provide a unified platform where users can legally stream millions of movies, TV series, and documentaries curated from diverse, high-quality alternative sources.

## ✨ Key Features
*   **No Trending Content:** Strictly free from current trending mainstream movies.
*   **Massive Library:** Guaranteed addition of 2,000,000 (2 Million) free, legal streaming links in each section.
*   **Universal Subtitles:** Built-in subtitle support for all languages across all media.
*   **Global Documentaries:** A dedicated, comprehensive collection of documentaries from every country in the world.
*   **Automated Data Fallback Mechanism:** Intelligent processing of missing data (like missing files or IMDb ratings) using automated fallback archives.

## 📡 Content Sources
All content is aggregated legally from the following platforms and categories:
*   TMDB (The Movie Database)
*   YouTube
*   Internet Archive
*   National Geographic (NatGeo)
*   Vice
*   Discovery Channel
*   BBC
*   Al-Jazeera
*   Oscar-Winning Masterpieces
*   Timeless & Classic Cinema
*   Various Alternative Media Outlets

## 🛠️ Installation & Setup (Developer Guide)

1.  **Primary Data Integration:**
    *   Extract and integrate the primary provided `.zip` file into the root directory. 
    *   This zip file contains the core database and streaming links.

2.  **Fallback Data Handling:**
    *   During runtime, the application will verify files and fetch IMDb ratings.
    *   **Conditional Fallback:** If a specific file or IMDb rating is missing from the primary zip archive, the system will automatically inject and utilize the secondary provided backup file to fill in the missing gaps.

## ⚖️ Legal Disclaimer
All media provided within **Chole Cholochitro** is sourced through legal, open-source, or publicly available free-to-stream APIs and archives. The platform does not host pirated content.
